### DarkHub - Update v1.1.9
**[+] Fixed & Improved Features**
 - **Windows 11 compatibility pass** - Updated the app target to .NET 8 Windows, refreshed the Windows 11 manifest compatibility, and fixed several APIs that were fragile on modern Windows builds.
 - **Fixed startup dependency detection** - Critical dependencies are now resolved from the application folder, while optional tools no longer block startup when missing.
 - **Restored legacy setup assets** - The installer now includes the recovered assets from the previous installation, including ffmpeg, yt-dlp, CPU-Z, GPU-Z, HWiNFO, DDU, SpaceSniffer and the DDU settings database.
 - **New Inno Setup installer flow** - Recreated the classic `DarkHub.Setup.exe` installer, with desktop/start menu shortcuts and .NET 8 Desktop Runtime bootstrap support.
 - **New signed release pipeline** - Added scripts for publish, installer creation and Authenticode signing through external certificates or certificate thumbprints.
 - **Removed bundled private certificate** - The old `DarkHub.pfx` is no longer shipped or tracked; release signing now expects a real external certificate.
 - **PDF conversion fixed** - Replaced the legacy `PdfiumViewer` package with `PDFtoImage` for .NET 8 compatible PDF rendering on Windows 11.
 - **File converter improved** - Hardened ffmpeg execution, temporary file cleanup, PDF to image export and raw/bin conversion paths.
 - **YouTube downloader improved** - Missing yt-dlp/ffmpeg now produces actionable messages instead of constructor-time failures, and transcript handling is safer.
 - **System monitor improved** - Windows 11 version detection, GPU/VRAM readings, RAM/disk/network fallbacks and benchmark guards were corrected.
 - **Optimizer tools improved** - Updated process priority, service management, startup management, DNS benchmarking, network optimization, registry cleaning, temp cleaning and Windows repair flows.
 - **High performance mode improved** - Fixed Windows 11 registry paths, Ultimate Performance power plan handling and command execution timeouts.
 - **Program uninstaller improved** - Reworked uninstall command parsing for quoted paths, MSI commands and System32/PATH resolution.
 - **Security and metadata tools improved** - Migrated PDF metadata handling to iText 7, hardened Advanced Security null handling and removed unsafe certificate installation behavior.
 - **Password manager improved** - Replaced predictable random generation for passwords and recovery words with cryptographic randomness.
 - **DLL injection service stabilized** - Cleaned Windows 11/nullability issues, WMI reads, executable path handling, export lookup and Authenticode validation for consented defensive workflows.
 - **Build quality improved** - Release build now completes with `0 Warning(s), 0 Error(s)`.
